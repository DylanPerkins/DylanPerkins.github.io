## Projects

### Project 1: [Documentation Website](https://dylanperkins.github.io/project_1)

**Competency**: Software Design & Engineering

**Description**: This project was created in October of 2023 as the official documentation website for a software development company that I was working for. It is a Docusaurus 3-based website that uses Node, is hosted on its own domain and utilizes .mdx files for documentation and blog pages. I refactored this project in early 2026 to implement lazy loading for images to improve performance and the user experience.

### Project 2: [8 Puzzle Solver](https://dylanperkins.github.io/project_2)

**Competency**: Algorithms and Data Structures

**Description**: This project was created basically by scratch for the purpose of this portfolio and I used a lot of inspiration from my CS-370 class that I took in early 2026. This is a Python and Jupyter Notebook based project that builds an agentic model using the A2C algorithm to solve the problem of the 8 puzzle, which is a sliding tile puzzle that consists of a 3x3 grid with 8 numbered tiles and one empty space. The purpose of this project was to demonstrate my understanding of algorithms and data structures, as well as my ability to implement them in a real world application that I was personally looking to solve.

### Project 3: [Anime Tracker](https://dylanperkins.github.io/project_3)

**Competency**: Databases

**Description**: This project was initially created in February of 2023 as a final project and capstone for a backend bootcamp. It is a Java-based application that uses Spring Boot and MySQL to create an API for users to review, rate and track anime and create anime lists. I refactored this project in 2026 to implement secure coding techniques like input validation, fixing an ownership exploit, add two new API endpoints that let users query by anime name and tags and a myriad of other improvements.
